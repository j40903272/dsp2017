===== FILES =====
corpus.txt是此次作業的語料檔案，編碼為big5
Big5-ZhuYin.map是文字跟注音的對照表，編碼為big5
separator_big5.pl將文字之間安插空白
Submit_Files 是繳交作業的資料結構
separator_big5.pl執行方法為

  ./separator_big5.pl < textfile > output

(使用前先執行chmod 755 separator_big5.pl)
